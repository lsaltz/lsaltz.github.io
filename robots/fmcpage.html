Follow-Me Car

I decided to build a robot car that follows me around like a puppy.

I first used Fusion360 to CAD it (render below), and am currently printing it.

I am using ball bearings as the front wheels, and two steppers for the back wheels.

Its brain is a Jetson Nano, and it uses a Waveshare motor hat.

I worked with OpenCV's HOG to detect me and move the motors based on my position in the frame.

See the code here

Ultimately, using OpenCV, the robot was able to detect and track me, and the motors moved accordingly.

If I were to take this project further, I would implement a parallel motor control system, as the motors for this project were controlled sequentially.
